# UnityAstcSupportDemo

## 简介
Unity根据硬件支持来选择加载ASTC还是ETC/PVRTC的demo。

## 运行
- File -> Build Settings... -> Android -> Switch Platform
- Tools -> Build Asset Bundles
- Play
